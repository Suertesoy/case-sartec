## Como usar este arquivo

1. Crie ou exporte cada imagem conforme a descrição acima
2. Salve na pasta `assets/media/` com o nome exato indicado
3. Quando a imagem existir, substitua o placeholder na `index.html`:
   - Localize o bloco com a classe `media-placeholder` correspondente
   - Substitua o div pelo elemento `<img>` com o `src` e `alt` corretos
   - Mantenha o `<p class="media-caption">` abaixo com a legenda sugerida
